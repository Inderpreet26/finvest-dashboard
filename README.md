# 💹 FinVest Dashboard

Full-stack finance & investment dashboard with microservices.

## 🏗️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python + FastAPI (7 Microservices)

## 🚀 Microservices
| Service | Port | Purpose |
|---------|------|---------|
| auth-service | 8001 | Authentication |
| portfolio-service | 8002 | Investment Portfolio |
| market-service | 8003 | Stock/Crypto Prices |
| news-service | 8004 | Financial News |
| analytics-service | 8005 | Charts & Analytics |
| watchlist-service | 8006 | Stock Watchlist |
| notification-service | 8007 | Price Alerts |
